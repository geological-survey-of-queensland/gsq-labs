3D Graph Tool


This web-based utility is a visualisation tool designed for graph data exploration. Built upon the React Force Graph library by Vasturiano, it serves the primary purpose of rendering data into interactive 3D force-directed graphs. Users have the capability to import graph data in standardized formats like JSON, JSON-LD, and TTL, and to interact with the visual representation through features such as zooming, panning, filtering and node inspection. The TTL files are converted to JSON-LD using Frogcat's Turtle to JSON-LD converter,

For some background around how GSQ is using this tool as a visual aid in the development of knowledge graphs, refer to a UQ/GSQ Webinar about knowledge graphs. Those who wish to delve deeper into the mechanics or modify the tool to suit specific needs can access the codebase on the project's GitHub codebase. Feedback from the community is welcomed to refine and enhance the functionality of this visualisation resource.

This tool is not supported or maintained and is to be used as-is and without warranty. It is undergoing continual development and occasional bugs and issues are likely to occur. The GSQ Labs source code (where this is hosted) can be downloaded from GitHub and run locally as required. Please see disclaimer disclaimer for further info.
